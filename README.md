# BookForge Studio

موقع ثابت جاهز للرفع على GitHub Pages لصناعة كتاب إلكتروني من أجزاء/بارتات نصية.

## المميزات
- رفع عنوان الكتاب، اسم المؤلف، وصف الكتاب، وصورة الغلاف.
- رفع مرجع ستايل على شكل صورة أو PDF لتحليل الألوان واقتراح ستايل قريب.
- رفع ملفات نصية متعددة دفعة واحدة: TXT / MD / DOCX.
- إضافة بارتات يدويًا، تعديلها، حذفها، وإعادة ترتيبها.
- معاينة حيّة للكتاب بشكل صفحات أنيقة.
- تصدير بصيغ: PDF / HTML / TXT.

## التشغيل
فقط افتح `index.html` أو ارفع الملفات إلى GitHub Pages.

## ملاحظة مهمة
الموقع يلتقط **الجو العام** من المرجع (ألوان/مزاج/تنسيق مقترح) لكنه لا ينسخ تصميم الكتاب المرجعي أو خطه بشكل حرفي 100%.
