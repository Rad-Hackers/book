BookForge Studio Ultimate

رفع الملفات إلى GitHub Pages:
1) فك الضغط عن الملف ZIP.
2) ارفع جميع الملفات كما هي إلى المستودع.
3) فعّل GitHub Pages من إعدادات المستودع.

أهم الميزات:
- رفع غلاف ومرجع تصميم
- تحليل Style DNA من صورة أو PDF
- تقسيم صفحات حقيقي ثابت الحجم
- صفحات تمهيدية: إهداء، حقوق، عن الكتاب، فهرس
- تنسيق الحوارات والاقتباسات والجمل المميزة
- مولد غلاف تلقائي
- حفظ المشروع محليًا واسترجاعه
- تنزيل كل صفحة كصورة
- تنزيل جميع الصفحات ZIP
- تصدير PDF من صور المعاينة نفسها لتجنب الصفحات البيضاء
- مقارنة بين المعاينة والنسخة النهائية
- ثيمات متعددة للروايات والكتب
- دعم TXT / MD / DOCX
